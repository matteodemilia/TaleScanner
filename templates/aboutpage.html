    Tale Scanner Overview

    What is Tale Scanner used for?

    Tale scanner is developed to help Speech and language pathologists analyze test results in a fast and efficient way.

    How is it made?

    We utilize spaCy which is a natural language processing python library.
    What spaCy analyzes is the type-token ratio, morphemes, number of words, unique words, and clauses.
    We also utilize a library called “Gramformer”. Gramformer is used to detect verb errors in the user input.

    Our goal

    Our main goal is to help Speech and language pathologists by limiting time spent analyzing test results.
    A lot of their time is taken counting the specific statistics in each test

    Acknowledgements of anomalies

    Tale scanner can only analyze English at the moment.
    We hope to expand the range of different types of languages users can analyze in the future.
    Please note that some check boxes take longer to analyze compared to others.

  • How do I use Tale Scanner?

    Input page

    Simply copy and paste text in the input box.
    Next check which boxes you would like to analyze for!

    Results page

    At the moment our results only count the amount that the user checks in the input page.
    It analyzes each sentence one by one.
    click the "+" to see more details about the chosen analysis, confidence rating and how the analysis got calculated.

  • How to Read Tale Scanner Results

    about how to read results

    How are results determined?

    The results are determined by algorithms our team made with the help of "spaCy" and “Gramformer”.

    How is confidence rating determined?


    Testing is done to ensure our system is accuratly analyzing the inputted text by the user.
    We first test each analysis check box by itself to verify functionaility.
    We then pair the check boxes with another. Lastly, we group them together to find out any potential
    issues with the analysis. Another important reason is to also check the runtime to warn users of which
    ones take longer.

    Confidence rating per results

    Having a high confidence rating means the results have high accuracy with no mistakes.
    Those with lower confidence ratings mean there are some errors in the analysis.
